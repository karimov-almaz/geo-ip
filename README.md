## Description

Web сервис для получения информации о гео координатах и местоположении (страна, город) по IP-адресу (v4, v6)

## Swagger

http://localhost:3000/docs

## Installation

```bash
$ npm install
```
build sdk
```bash
$ npx nest-sdk-generator sdk/sdk-generator.json
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
